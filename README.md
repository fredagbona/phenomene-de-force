# 🐅 PHÉNOMÈNE DE FORCE | Landing Page High-Conversion

> **"Le monde s'en fout de toi. Lève-toi."** > Une refonte d'élite pour une marque d'élite. 

Ce projet est une proposition de refonte UI/UX pour la marque **Phénomène de Force**. L'objectif est de combler le décalage entre la puissance du message social (320k+ followers) et l'expérience de vente sur le web.

## 🎯 Objectifs du Projet
- **Boost de Conversion :** Design Dark Mode agressif pour augmenter la valeur perçue.
- **Autorité Massive :** Mise en avant stratégique des preuves sociales (Facebook, Insta, Threads).
- **Performance Mobile :** Architecture Next.js pour un chargement instantané (LCP < 1.2s).
- **Immersion :** Intégration de vidéos de motivation optimisées pour l'engagement.

## 🛠 Tech Stack
- **Framework :** [Next.js 14](https://nextjs.org/) (App Router)
- **Styling :** [Tailwind CSS](https://tailwindcss.com/)
- **Animations :** Framer Motion (Scroll reveal & Compteurs)
- **Hébergement :** [Netlify](https://www.netlify.com/)
- **Fonts :** Bebas Neue (Titres) & Roboto (Body)

## ⚡️ Optimisations Multimédia
Pour garantir une fluidité maximale malgré l'utilisation de vidéos :
- Vidéos compressées (< 3Mo par unité) sans perte de netteté.
- Suppression des pistes audio pour réduire le poids.
- Utilisation des attributs `muted`, `playsinline` et `loop` pour un rendu natif fluide.
- Stratégie de *Lazy Loading* pour ne charger les vidéos qu'au scroll.
